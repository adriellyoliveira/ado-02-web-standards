## Sobre o projeto

Este repositório foi desenvolvido como continuidade da ADO 01, com o objetivo de tornar a interface totalmente responsiva para diferentes tamanhos de tela.

A atividade consiste na adaptação das seções Navbar, Hero Header e Footer para dispositivos mobile, tablet e desktop, utilizando exclusivamente HTML e CSS puro, sem o uso de frameworks.

O projeto segue a abordagem mobile first, aplicando media queries com `min-width` para construção progressiva do layout responsivo.

---

## Objetivos

* Aplicar conceitos de responsividade em interfaces web
* Praticar a abordagem mobile first
* Adaptar layouts para diferentes dispositivos
* Trabalhar com Flexbox e CSS Grid
* Desenvolver navegação responsiva com menu colapsável
* Reforçar boas práticas de organização e estruturação CSS

## Responsividade

O projeto utiliza breakpoints para adaptação da interface em diferentes resoluções:

* Mobile: estrutura base do projeto
* Tablet: `min-width: 768px`
* Desktop: `min-width: 1280px`

As adaptações incluem:

* Menu responsivo (hamburger menu)
* Reorganização de layouts em colunas
* Ajustes de espaçamento e alinhamento
* Reestruturação do footer para telas menores
